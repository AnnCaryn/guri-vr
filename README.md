
# Guri VR

Code for [GuriVR.com](https://gurivr.com)

## Requirements

- Mongodb
- Node.js >= v4

## Installation

- `$ cp config.sample.json config.json`
- Adapt the configuration to your environment
- `$ npm install`
- `$ npm start`

## Frontend Development
- `$ cd client`
- `$ npm install`
- `$ npm start`
